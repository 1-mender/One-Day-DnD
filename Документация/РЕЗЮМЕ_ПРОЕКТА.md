## 🎯 ЧТО ТАКОЕ ONE-DAY-DND?

**One-Day-DnD** — это веб-приложение для проведения сессий Dungeons & Dragons в локальной сети (LAN) без интернета.

- **Мастер (DM)** сидит с ноутбуком и открывает `http://localhost:3000/dm`
- **Игроки** подключаются через Wi-Fi по QR коду или ссылке
- **Все действия** синхронизируются в реальном времени через WebSockets
- **Данные** сохраняются в SQLite БД на машине DM

## 🎮 КАК ИСПОЛЬЗОВАТЬ

### Сценарий 1: Локальная сессия (1 ПК)
1. Запустить `npm run dev`
2. Открыть `http://localhost:3000/dm` (Мастер)
3. Открыть `http://localhost:3000/player` (Игрок)
4. Переключаться между вкладками

### Сценарий 2: WiFi сессия (несколько ПК)
1. На ПК Мастера: `npm run dev`
2. Найти IP: `ipconfig` (Windows) или `ifconfig` (Linux)
3. Игроки открывают: `http://<IP>:3000/player`
4. Сканируют QR или вводят код

### Сценарий 3: Production сервер
1. `npm run build`
2. `npm run start`
3. Доступ через reverse proxy (Nginx)
