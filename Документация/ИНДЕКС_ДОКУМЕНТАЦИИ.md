## 🚀 БЫСТРЫЕ КОМАНДЫ

### Установка и запуск

```bash
# Полная установка
npm install && npm --prefix server install && npm --prefix client install

# Запуск в режиме разработки
npm run dev

# Открыть в браузере
http://localhost:3000/dm        # Мастер
http://localhost:5173/dm        # Vite dev
```

### Проверка и тестирование

```bash
# Быстрая проверка перед сессией
npm run preflight

# Полная верификация
npm run verify

# E2E тесты
npm run e2e

# Unit тесты
npm run test
```

### Production

```bash
# Собрать для production
npm run build

# Запустить production версию
npm run start
```
